Migrando dados entre instalações django

Tire os dados do sqlite e mande para um mysql

Precisei levar alguns dados que estavam em sqlite3, numa instalação local de django, para uma instalação em produção, em mysql. Uma simples pesquisa no google me levou a este post:

﻿Django Dumpdata to Fixture for Initial Data

A extensão do nome do arquivo gerado não pode ser qualquer um. Por exemplo, para pegar todos os dados da app city:


cd /dir/da/minha/app
python manage.py dumpdata city > /tmp/city.json

copia o arquivo pro servidor, e lá:


cd /dir/da/minha/app
python manage.py loaddata /tmp/city.json

RTFM, pois tem muito mais informação por lá:

  • dumpdata

  • loaddata
